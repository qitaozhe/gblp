# Contributing to GBLP

First, thank you for considering contributing! GBLP is a speculative proposal for a future global monetary system. We welcome:

- Technical feedback (cryptographic flaws, economic inconsistencies)
- Political feasibility insights (especially from non-Western perspectives)
- Clarifications, translations, and use-case scenarios

## How to Contribute

1. **Open an issue** – for questions, critiques, or suggestions.
2. **Fork & pull request** – for direct changes to documents (typos, clarifications, new sections).
3. **Start a discussion** – in GitHub Discussions if you want to debate a core assumption.

## Style Guidelines

- Use plain, accessible English (or good-faith machine translation from any language).
- Avoid ad hominem attacks – debate ideas only.
- When proposing a change to a core mechanism, include a “trade-offs” section explaining what you would break.

## Review Process

All contributions will be reviewed by the current maintainer(s) (initially, the original author). We aim for a lightweight, transparent process.

Thank you for helping build a more just digital future.
