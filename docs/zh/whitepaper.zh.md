# 地球基本钱包技术白皮书（概要）– GBLP v0.1

## 1. 身份与钱包

- **DID + 生物特征 + 社会恢复**。
- 用户离线生成DID，生物特征哈希本地存储。可指定5位见证人用于密钥恢复。
- 无证件人士：经三位社区见证人获得临时DID，两年后转为永久。

## 2. 生存线及基本额度

- 生存线 = 每月食品、水、住所、基础医疗、基础通信的最低成本。
- 区域数据来自FAO、WHO及本地统计。
- **基本额度 = 生存线的80%**（低于生存线）。

## 3. 倒卖惩罚机制

- 基本额度仅限指定商户类别码（食品、水电、基础药品）使用。
- AI风控检测异常模式。
- **惩罚阶梯（封顶6个月）**：
  - 第1次：当月额度清零，半数转举报者；停发当月剩余部分。
  - 第2次：停发1个月
  - 第3次：停发2个月
  - 第4次：停发4个月
  - 第5次及以上：停发6个月（封顶），期满后重置计数。
- 申诉：仲裁庭（3名技术专家+2名公民观察员）。

## 4. 实物援助凭证

- 援助方承诺捐赠物资（全新，或危机期间8成新以上）。
- 随机抽检由3个独立节点（NGO、大学）执行。
- 验证通过后铸造NFT凭证，分发至接收者基本钱包。
- 凭证在本地仓库兑换后销毁。
- 违规：虚假申报退回并承担运费；三次违规暂停实物援助资格12个月。

## 5. 抗国家封锁技术

- 离线P2P同步（蓝牙网状网络）和卫星广播。
- 政府断网后，EBW客户端自动切换离线模式。
- “保护性暂存”：某国90%连接中断超过48小时，钱包冻结但保留；恢复连接后补发所有欠额。
- 对封锁政府的制裁见章程。

## 6. 全球结算币（GSC）概要

- 铸造：任何人可锁定经验证的实物篮子（通过物联网仓单）铸造GSC。
- 赎回：销毁GSC获得对应实物。
- GSC供应量随实际经济活动自动调节，无任意通胀。
- 央行联盟节点验证每笔铸造/赎回，共识机制为BFT。

## 7. 兑换池（区域币 ↔ GSC）

- 每个区域运行一个以GSC作为抵押的流动性池。
- 汇率 = 基于该区域CPI、失业率、财政赤字、池深度的算法。
- 每笔兑换扣除0.1%“团结税”用于支持基本钱包。
