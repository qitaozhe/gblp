# Governance Charter (Draft) – GBLP

Note: Full English translation in progress. Core principles summarized below. For detailed articles (15 articles), please refer to the [Chinese version](zh/charter.zh.md).

**Summary**:
- Three-chamber voting: Regional Representatives (1 per state), Technical Experts (21), Citizen Observers (advisory).
- Supermajority required for basic unit formula changes, sanctions, GSC basket adjustments.
- Sanctions: 3-year EBW suspension for leaders who block EBW access; country exchange pool throttled.
- Withdrawal possible with 6-month notice.

See [中文版章程](zh/charter.zh.md) for complete draft.
